# Youtube Frontend

Youtube Api for data processing


## Getting Started

### Prerequisites

- React
- Axios

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Hudeh/youtube-api-frontend.git
    cd youtube-api-frontend
    ```

2. **install**

    ```bash
    npm install
    ```

3. **start**

    ```bash
    npm start
    ```
